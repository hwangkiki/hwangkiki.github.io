前言

作为一个小白，一直想要一个自己的博客平台，之前是使用的wordpress基本只要设置好了环境就是一键部署，网站依赖于服务器，

但是自己想了一下不仅仅是需要一个博客而来创造博客，搭建的过程也是一个学习的过程，大概了解前端的知识，而且hexo可以把网站托管到github上这样就算服务器到期了也可以继续使用，所以这样就开始了我敲代码建站之旅。

所有处理方式基于Mac，win可以此类推
